Scenario 1: Developer A starts developing an Object and nobody else is developing it until it is deployed to PROD

;

Resolution State in GitHub
  • Developer A creates a branch dev_a and commits those changes to the branch
  • Developer A pushes this branch from the local repository,to the repository hosted on GitHub Enterprise
  • Developer A opens a Pull Request from branch dev_a targeting branch master, triggering a review among peers and any automated builds set up on Pull Request creation.
  • Developer A merges branch dev_a into branch master after review and closes the Pull Request
